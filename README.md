# Advanced Programming Libraries – Rust Projects

This repository contains a collection of Rust projects developed as part of the **Advanced Programming Libraries** course during my studies.

## Overview

The goal of this repository is to showcase practical implementations of concepts covered in the course, including:

* Working with advanced Rust libraries and ecosystems
* Applying idiomatic Rust patterns and best practices
* Exploring performance-oriented and safe system design
* Experimenting with different programming paradigms in Rust

Each project focuses on a specific topic or library, providing hands-on experience and examples that demonstrate how to effectively use Rust in real-world scenarios.

## Structure

The repository is organized into multiple independent projects. Each project:

* Is self-contained
* Includes its own `Cargo.toml` configuration
* Can be built and run separately

## Requirements

* Rust (latest stable version recommended)
* Cargo (comes bundled with Rust)

## Getting Started

To run a specific project:

```bash
cd <project_name>
cargo run
```

To build without running:

```bash
cargo build
```

## Purpose

This repository serves as:

* A learning resource for advanced Rust programming
* A reference for working with various Rust libraries
* A portfolio of coursework and experiments
